以金鑰使用 SSH
====================
ssh-keygen -t rsa
    # 可從 linux shell 或 git bash for Windows 執行該指令

產生 id_rsa、id_rsa.pub 兩個檔案
    # Linux 系統裏檔案位置在 ~/.ssh，Windows 系統裏檔案位置在 path\to\usrs\xxxx\.ssh
    # id_rsa 檔案為私鑰，id_rsa.pub 檔案為公鑰

將私鑰留在預定位置

將公鑰送至執行服務的伺服器
