以金鑰使用 SSH
====================
ssh-keygen -t rsa
    # 可從 linux shell 或 git bash for Windows 執行該指令

產生 id_rsa、id_rsa.pub 兩個檔案
    # Linux 系統裏檔案位置在 ~/.ssh，Windows 系統裏檔案位置在 path\to\usrs\xxxx\.ssh
    # id_rsa 檔案為私鑰，id_rsa.pub 檔案為公鑰

將私鑰留在預定位置

將公鑰送至執行服務的伺服器




SSH 安全性設定
====================
1. SSH 預設埠號是 22，改成其他的埠號。
2. 禁止以 root 帳號登入
   /etc/ssh/sshd_config => PermitRootlogin no
3. 只准特定帳號或群組登入 SSH
   /etc/ssh/sshd_config => AllowUsers my_name | AllowGroups my_group 
4. 帳號是空密碼, 禁止登入:
   /etc/ssh/sshd_config => PermitEmptyPasswords no
5. 只開啟 Protocol 2 版本
   /etc/ssh/sshd_config => Protocol 2
6. 使用金鑰登入
    a. 產生金鑰
       ssh-keygen -t rsa -b 4096
    b. 將 id_rsa.pub (public key) 上載到伺服器 ~/.ssh/authorized_keys/ 內
       ssh-copy-id -i ~/.ssh/id_rsa.pub user_name@machine
    c. 回到伺服器端，開啟 SSH 設定檔 /etc/ssh/sshd_config，將以下一行的注解取消，改成：
       AuthorizedKeysFile %h/.ssh/authorized_keys
    d. 重新啟動 SSH 伺服器:
       systemctl restart sshd
