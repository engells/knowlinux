Apache 虛擬主機
====================

編輯 /etc/apach2/sites-available/www.engells.idv.conf
<VirtualHost *:80>
	ServerAdmin admit@engells.idv
	ServerName engells.idv
	ServerAlias www.engells.idv
	DocumentRoot /webs/engells
	ErrorLog /webs/logs/engells/www.error.log
	CustomLog /webs/logs/engells/www.access.log  combined
	<Directory "/webs/engells">
		AllowOverride All
		Order Allow,Deny
		Allow from All
	</Directory>
</VirtualHost>

<VirtualHost *:80>
	ServerAdmin admit@engells.idv
	ServerName engells.idv
	ServerAlias acct.engells.idv
	DocumentRoot /webs/engells
	ErrorLog off
	CustomLog /dev/null combined
	<Directory "/webs/engells">
		AllowOverride All
		Require all granted
	</Directory>
</VirtualHost>




Apache 目錄權限
====================
http://xxx.xxx.xxx.xxx/test
編輯 /etc/apach2/apach2.conf
<!-- 設定虛擬目錄為 localhost/test/，實際檔案位置在 /home/franklin/test/ -->
Alias /test/ "/home/franklin/test/"
<Directory "/home/franklin/test/">
    Options FollowSymLinks
    AllowOverride None
    Require all granted
</Directory>
