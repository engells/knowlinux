VMware Server虛擬

  • How to install vmware 2.x, following the commands below:

    sudo apt-get install linux-headers-`uname -r` build-essential xinetd
    sudo ./vmware-install.pl
    sudo passwd root

  • How to manage vmware Server 2.0, through web interface with url:
    • https://127.0.0.1:8333
    • http://127.0.0.1:8222
  • 最佳化VMWare
    • 將預設虛擬網卡從AMD PCNet PCI換成Intel Pro1000，傳輸速度可由400kpbs提高至3Mpbs
    • 關閉虛擬機器後，進入虛擬機器檔所在目錄，以文字編輯器打開.vmx檔
    • 在.vmx檔的Ethernet節段第一行前插入ethernet0.virtualDev = e1000
    • 下載Intel網卡驅動程式
    • 存檔後重新啟動虛擬機器，在虛擬機器中刪除AMD網卡，並安裝Intel網卡。
    • 再度關閉虛擬機器並重新啟動一次，在虛擬機器中進入Windows的設定>網路連線> 區域 連線 >內容>一般>網卡的進階設定，將連結速度及雙工選項設為自動偵測
