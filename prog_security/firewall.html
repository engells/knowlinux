iptables

將 iptables* 複製到 /opt/security/iptables 目錄下，並設定為可執行屬性
編輯/etc/init.d/rc.local檔案，在檔尾加入"bash /opt/security/iptables/iptables.rule"

Firestarter

Auto assign authoirty on Firestarter when booting Ubuntu:
  • Run command: sudo gedit /etc/sudoers
  • Add the following line into the file of /etc/sudoers  
    username ALL= NOPASSWD:/usr/sbin/firestarter => username dtands for your account
  • Add the follwing line at System > Preferences > Startup Applications
      sudo firestarter --start-hidden
