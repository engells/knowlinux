tmux 指令
====================
C-a 方向鍵			# 切換 window
C-a [0...9]		# 切換 window
C-a K 				# 關閉 window
C-d 					# 關閉 tmux panel

tmux or tmux new -s session_name						# 新增 session
tmux attach or tmux attach -t session-id		# 連回 session
tmux rename-session -t 0 session_name				# 將 session 0 命名為 session_name
tmux detach
tmux ls
tmux kill-session -t 0											# 刪除 session
tmux kill-server														# 刪除 tmux server

tmux split-window														# 垂直分割畫面
tmux split-window -h -p 40									# 水平分割畫面，下方高度為終端機高度之 40%

tmux new-window -t session_name:0 -n 'window_name0' '/usr/bin/irssi'			# 在 session_name 建立編號第 0 號 window，並執行 /usr/bin/irssi 指令 
tmux new-window -t session_name:1 -n 'window_name1' 'ssh w1.example.com'
tmux new-window -t session_name:2 -n 'window_name2' 'ssh w2.example.com'
tmux select-window -t session_name:1				# 選擇 session_name 編號第 1 號 window
tmux select-pane -t 1												# 選擇編號第 1 號 panel

conf commands
  set-option -s       # sever option
  set-option          # session option
  set-window-option   # window option
  set-option -g       # global option




比照 Screen 及 Tmux 快速鍵操作方式
====================
Tmux 原生快速鍵(prefix: Ctrl + b)
  Panel
    <prefix-%>      水平切割 panel
    <prefix-">      垂直切割 panel
    <prefix-x>      關閉 panel
    <prefix-方向鍵>  切換 panel
  Window
    <prefix-c>      產生新的 window
    <prefix-&>      關閉目前 window
    <prefix-p>      切換至前一 window
    <prefix-n>      切換至後一 window
    <prefix num>	  切換至指定編號 window
  Session
    <prefix>d      離開目前 session
    <prefix>D      離開指定編號 session



編輯設定檔： ~/.tmux.conf，加入下列檔案內容
	/usr/share/doc/tmux/examples/screen-keys.conf
	/usr/share/doc/tmux/examples/vim-keys.conf

	C-a ? 看說明 (C-a 代表 Crtl＋a； M 代表要先按 Esc 鍵)
	C-a t 時鐘, 現在時間
	C-a q 可以看到 pane 數字 - 若有 pane 比較容易懂

	C-a Ctrl按著 + 上下左右, 可以調整 pane 大小
	C-a s 橫切
	C-a v 直切 (或 C-a %)
	C-a C-o 調換 pane 的 位置
	C-a 上下左右 跳到上下左右的 pane

	C-a Esc 1 直的切割視窗排列 (M-1)，假設已先產生四格 pane，以下同。
	C-a Esc 2 橫的切割視窗排列 (M-2)
	C-a Esc 3 上面一個大的 pane + 下面三個 pane (M-3)
	C-a Esc 4 左面一個大的 pane + 右面三個 pane (M-4)
	C-a Esc 5 四個 pane 各 1/4 (M-5)




Screen 快速鍵
====================
C-a c 或 C-a C-c 開啟新的視窗分頁，並同時切換到視窗分頁
C-a n 或 C-a C-n 或 C-a (space) 切換到下一個視窗分頁（0->1 1->2 ...）
C-a p 或 C-a C-p 切換到上一個視窗分頁（1->0 2->1 ...）
C-a C-a 切換到上一個顯示過的視窗分頁（不是照順序切換）
C-a (0,1..9) 切換到第 (0,1..9) 個視窗分頁
C-a w 或 C-a C-w 會列出目前所開啟的視窗分頁
	0$ tcsh 1$ tcsh 2*$ tcsh 3$ tcsh：有星號 "*" 表示目前顯示中的子視窗，之後的 tcsh 代表子視窗的 title ，預設是開啟時的 shell 名稱 

C-a K 關閉所有的視窗分頁並退出 screen
C-a ' 或 C-a " 會出現 "Switch to window:" 字樣，輸入號碼後就可切到該視窗分頁

C-a C 清除目前的視窗分頁內容
C-a d 或 C-a C-d 脫離（detach）目前的 screen ，並放到背景執行（不管開了幾個視窗分頁）
C-a D D 強力脫離，除了放到背景執行外，並自動 logout

C-a C-g 虛擬嗶聲（visual bell）的切換，可以在畫面上顯示訊息代表一般的嗶聲
C-a H 紀錄目前視窗分頁所有顯示過的東西（檔名為 screenlog.n n 為視窗號碼）
C-a i 或 C-a C-i 顯示目前視窗分頁資訊
	(11,24)/(80,24)+1000 +(+)flow -ins -org +wrap +app -log -mon +r G0 [BBBB]

C-a l 或 C-a C-l 重繪視窗分頁，如果有字顯示亂掉了可以用這個指令
C-a m 或 C-a C-m 或 C-a (enter) 可以顯示最近一次顯示過的訊息
C-a t 或 C-a C-t 顯示系統的時間及主機名稱還有負載
C-a v 顯示版本資訊
C-a x 或 C-a C-x 如果要暫時離開目前位置又不想先 logout ，那可以用這個來鎖住 screen
C-a S 將營幕畫面暫停
C-a q (C-a S 暫停) 的 回復
C-a ? 線上求助畫面

C-a [ 進入 複製/卷軸 模式（只有在目前視窗才算，其他視窗照常）基本上跟 vi 很像：
	h, j, k, l 分別為向左，向下，向上，向右移動一格（或是一行）
	0 移到該行最左邊
	^ 與 $ 分別移到該行最左邊及最右邊的非空白字元
	w 以字為單位往前移動並移到字首
	b 以字為單位往後移動並移到字首
	e 以字為單位往前移動並移到字尾
	C-b 向上捲一頁
	C-f 向下捲一頁
	C-u 向上捲半頁
	C-u 向下捲半頁
	/ 與 ? 向下或向上尋找字串
	(space) 第一次按下到第二次按下的區域會被複製到暫存區內
	(ESC) 離開 複製/卷軸 模式
接著按下 C-a ] 便可以把剛剛存到暫存區內的內容複製出來。 




tmux 搭配 tmuxifier
====================
建立 tmuxifier
----------
git clone https://github.com/jimeh/tmuxifier.git ~/.tmuxifier


配置 tmuxifier
----------
編輯 ~/.profile 加入設定，讓 tmuxifier 指令自動列入 $PATH 環境變數。
	[[ -s "$HOME/.tmuxifier/init.sh" ]] && source "$HOME/.tmuxifier/init.sh"
	export EDITOR=vim


建立 tmuxifier 的視窗分割設定
----------
tmuxifier new-window window_layout_name
	window_root "~/path/to"		指定某專案的路徑作為 Shell 的 Working Directory
	new_window "Example"		設定標題
	split_h 50		建立水平分割 50% 寬度
	split_v 50		建立垂直分割 50% 高度
	run_cmd "vim" 0		分別在編號 0~2 三個 panel 跑指令
	run_cmd "date" 1
	run_cmd "htop" 2
	select_pane 0		將游標切換到 panel  0
	send_keys "iI Love Vim"			對作業中 panel 模仿鍵盤操作，送出編輯或操作指令，此例是對 viim 傳送 iI Love Vim 鍵盤操作


載入視窗分割設定
----------
執行 tmux 指令後，再執行 tmuxifier load-window window_layout_name 指令
視窗分割檔案：${tmuxifier_dir}/layouts


解決顏色錯誤
----------
編輯 ~/.bashrc，加入下列內容：
	export PATH="${tmuxifier_dir}/bin:$PATH"
	export TMUXIFIER_TMUX_OPTS="-2" => 強制使用 256 色
	eval "$(tmuxifier init -)"

編輯 ~/.tmux.conf
	set -g default-terminal "screen-256color"

Ref
	https://aranhase.github.io/2014/08/11/vim_tmux_tmuxifier_setup_with_konsole.html




Tmux 簡介
====================
tmux = Terminal Multiplexer = Σ sessions = Σ windows = Σ panes => each pane run a terminal instance

執行 tmux 指令會建立一個新的 tmux session，終端機畫面底部多了一條狀態列。狀態列是 tmux 很重要的部分。除了顯示當前的視窗（在左邊）它同時也會顯示一些關於系統的資訊像是時間（在右邊）。狀態列還可以依照需求客製，例如：顯示行事曆，電量等等。

設定檔 ~/.tmux.conf，參考設定檔：	/usr/share/doc/tmux/examples/screen-keys.conf，/usr/share/doc/tmux/examples/vim-keys.conf
